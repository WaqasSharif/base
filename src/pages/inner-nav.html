*************************************
section Name: Left Nav - left-nav-1
NOTE: to make links centered aligned please use 'center' class on div 'sec-left-nav'
NOTE: to make links right aligned please use 'right' class on div 'sec-left-nav'
NOTE: please use 'font-family-heading' class on div sec-left-nav' to make fonts family heading
NOTE: please use 'with-box' class on div 'sec-left-nav' to wrap links in a box
NOTE: if box needed border then use 'border' class on div 'sec-left-nav' to wrap links in a box
NOTE: to make box radius please use 'radius' class on div 'sec-left-nav'
NOTE: to put some animation please use 'animate-top' on div 'sec-left-nav'
NOTE: please use 'with-lines' on div 'sec-left-nav' if links needs lines as separator
NOTE: COLORS
NOTE: by default link color is gray, you can put 'dark' for the black links
NOTE: hovers link colors: please use 'white', 'anchor' 'default', 'primary' OR 'secondary' class on div 'sec-left-nav' for desired hover color combination
NOTE: hovers link background: please use 'hover-bg-light', 'hover-bg-default' 'hover-bg-primary', 'hover-bg-secondary' class on div 'sec-left-nav' for desired hover color combination
NOTE: BACKGROUNDS
NOTE: if we have choosen option 'with-box' then we can put 'default-bg', 'primary-bg' OR 'secondary-bg' to make backgrounds option for the box.
*************************************
NOTE: small-heading is optional - we can add or remove if needed NOTE: if small-heading don't need underline just remove class 'underline' on div 'small-heading' NOTE: please use 'with-heading-bg' class on div 'sec-left-nav' AND 'primary' or 'secondary' or 'default' on div 'small-heading' for desired background color of small heading
*************************************

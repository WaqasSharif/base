section Name: headings
NOTE: you can put any heading h1 to h6 inside 'sec-title' div and it will works
NOTE: COLORS - we have four colors 'default', 'primary', 'secondary' and 'light' please put respective class on div 'sec-title' for desired color
NOTE: SIZES - desktop/mobile -> h1 92/48 - h2 60/40 - h3 52/32 - h4 40/32 - h5 24/20 - h6 18/16
NOTE: UTILITY CLASSES - put 'bold' OR 'bolder' class on div 'sec-title' for bold/bolder text
put 'uppercase' class on div 'sec-title' for uppercase heading
Please use 'letter-spacing-0', 'letter-spacing-1', 'letter-spacing-2', 'letter-spacing-3', 'letter-spacing-4' class on div 'sec-title' for desired letter-spacing between letters put 'font-base' class on div 'sec-title' if headings needs to be in secondary font-family
